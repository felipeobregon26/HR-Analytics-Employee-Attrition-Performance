# Datos del Proyecto

## Dataset: IBM HR Analytics Employee Attrition & Performance

El archivo CSV no está incluido en este repositorio por razones de buenas prácticas.

## Cómo descargar el dataset

1. Ve a Kaggle: https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset
2. Descarga el archivo: WA_Fn-UseC_-HR-Employee-Attrition.csv
3. Colócalo en esta carpeta (data/) con el nombre exacto:
   WA_Fn-UseC_-HR-Employee-Attrition.csv

## Descripción del dataset

- Filas: 1.470 empleados
- Columnas: 35 variables
- Variable objetivo: Attrition (Yes/No)
- Fuente original: IBM Watson Analytics
- Licencia Kaggle: Open Database License (ODbL)
